## 1. Creating a DataFrame

### Purpose

Used to create tabular data for analysis and processing.

### Method

```python
import pandas as pd

data = {
    'Name': ["Ali", "None", "Ahmed", "Ayesha", "Usman"],
    'Age': [20, 35, 19, 19, 20],
    'Department': ["CS", "CS", "CS", "AI", "SE"],
    'GPA': [3.5, 2.6, 3.4, 3.2, 2.9]
}

df = pd.DataFrame(data)
```

Key insight
• DataFrame is the core pandas object
• Columns can be numeric or categorical

## 4. Handling Missing Data

### Checking Missing Values

```python
df.isnull()
df.isnull().sum()
```

### Filling Numeric Columns with Mean

```python
df['Age'] = df['Age'].fillna(df['Age'].mean())
df['GPA'] = df['GPA'].fillna(df['GPA'].mean())
```

### Filling Categorical Columns with Mode

```python
df['Name'] = df['Name'].fillna(df['Name'].mode()[0])
df['Department'] = df['Department'].fillna(df['Department'].mode()[0])
```

### Interpolating Numeric Data

```python
df['Age'] = df['Age'].interpolate(method='linear')
df['GPA'] = df['GPA'].interpolate(method='linear')
```

Strategic note
• Interpolation works only on numeric columns
• Avoid inplace with chained assignment
